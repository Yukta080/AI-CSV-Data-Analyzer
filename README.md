📊 AI CSV Data Analyzer

An AI-powered data analysis application built with **Python**, **Streamlit**, **Pandas**, and **Matplotlib** that enables users to upload CSV files, 
explore datasets, generate visualizations, and gain actionable business insights.

🚀 Features
* Upload any CSV dataset
* Preview uploaded data
* Display dataset information
* Perform automatic data cleaning
* Generate summary statistics
* Create interactive charts and visualizations
* Analyze sales, profit, and trends
* AI-ready architecture for natural language queries
* Simple and user-friendly interface

🛠️ Tech Stack
* Python
* Streamlit
* Pandas
* Matplotlib
* OpenAI API *(Coming Soon)*
* Git & GitHub

📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/AI-CSV-Data-Analyzer.git
```

Move into the project folder:

```bash
cd AI-CSV-Data-Analyzer
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
